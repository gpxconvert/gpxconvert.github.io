Disclaimer: This tool is for guidance only. It uses publicly available 2024/25 UK rates and assumptions listed below. Always check HMRC for definitive rules. Update rates in the code for 2025/26 when confirmed.

Assumptions (editable in code)

  • Income Tax (England/Wales 2024/25): Personal Allowance £12,570 (tapered £1 lost per £2 over £100,000), 20% basic up to £37,700 taxable, 40% to £125,140, 45% above. (Scottish bands not included here.)
  • NI (PAYE 2024/25): Employee Class 1 ~8% on £12,570–£50,270 and 2% above. (Set in code; adjust if rates change.)
  • NI (Self‑Employed 2024/25): Class 4 ~6% on £12,570–£50,270 and 2% above. Class 2 optional input as flat weekly amount.
  • Student Loans: 9% above plan threshold (Plan1/2/4/5), Postgrad 6%. Thresholds set in code.
  • Pension: Percentage of gross (relief‑at‑source assumed).

You can extend for Scottish rates, salary sacrifice, childcare vouchers, benefits‑in‑kind, etc.

FAQ
